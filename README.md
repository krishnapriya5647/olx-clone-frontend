# OLX Clone (Frontend)

A frontend-only OLX style classifieds marketplace built with React. Users can browse listings, search with filters, view product details, post ads (saved locally), and manage favorites.

## Features
- Listings feed (Home)
- Search with filters, sorting, pagination
- URL query params support (shareable search links)
- Product details page with image gallery
- Post Ad and My Ads (stored in localStorage)
- Favorites (stored in localStorage)
- Smooth Back button behavior (avoids too many history entries)

## Pages
- `/` Home
- `/search` Search results with filters
- `/product/:id` Product details
- `/post` Post a new ad
- `/my-ads` Manage your posted ads
- `/favorites` Saved favorites

## Tech Stack
- React
- JavaScript
- React Router
- HTML, CSS
- localStorage for persistence
